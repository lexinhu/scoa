# SCOA学分管理系统
一个可以导入学生成绩并查询到该学生每个课程分类总修得的学分，
查看详细还可以看到学生的成绩信息。

采用的技术栈

SpringBoot,Vue,ElementUI,Mybatis,Mybatis-plus

持续开发中。

- [x] 数据库设计
- [x] 登录Login
- [x] 个人信息读取
- [x] 前后端分离跨域问题
- [x] 学分展示
- [x] 成绩展示
- [x] 优化体验
- [x] Ex表格导入
- [x] 数据自动查重
  - 对表格录入的成绩信息查重
  - 对低于60分的同学过滤
- [x] 分页查询
